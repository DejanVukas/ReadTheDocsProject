DWDM grids
+++++++++++++

Fixed grid 100GHz
====================

- 100GHz spaced center-frequencies
- 44 channels in C-band

Fixed grid 50GHz
====================

- 50GHz spaced center-frequencies
- 88/96 channels in C-band

Flex grid 6.25GHz
====================

- Center frequencies are at 6.25GHz increments
- Width of each channel is N x 12.5GHz
